# -HomeTraining-

# 개요
```
1. 코로나 19의 확산으로 인한 외부 외출의 제한으로 집에서 쉽고 편하게 운동하기 위함

2. 바쁜 삶을 사는 현대인들의 올바른 스트레칭이나 운동 방법 및 운동 활동의 향상을 위해 개발

3. 자신이 운동한 종류 및 횟수의 반복 수에 따라서 칼로리를 계산하고 그 칼로리를 시각적인 그래프 형식으로 나타냄
```
